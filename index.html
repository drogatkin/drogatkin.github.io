Privacy policy for Android apps Is Genuine and Atjeews

Privacy policy for Android app Is Genuine

The application requests access to your camera to display its resolution. The application doesn't store, transfer or any pther way distribute the resolution information besides of showing it to a current phone operator.

If you do not want to see your phone camera resolution, then avoid to download and run Is Genuine

Privacy policy for Android app Atjeews

The application requests access to different features of your phone as camera, phone, contacts and etc to allow web applications using the features. Atjeews is an application server and doesn't use any requested permissions for itself.

Therefore consult with your web applications vendor regarding their privacy policies. Or do not use any 3rd party web applications.

Please note that JSP application I provide as a demo (a separate download from web), uses contacts information just for a demonstration purpose. It doesn't store it or transmit. Generally you run it in a source code and can confirm for yourself.
